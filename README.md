# Excel_Vrinda_Store_Analysis_Dashboard
End-to-end Excel data analytics project with an interactive dashboard for Vrinda Store sales performance analysis.

Project Overview
This project presents an interactive Excel dashboard created to analyze the annual sales performance of Vrinda Store (2022).
The dashboard provides meaningful business insights by visualizing sales trends, customer behavior, order status, and channel-wise performance using Excel tools.
